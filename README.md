# BillStack - Inventory Billing SaaS

BillStack is a full-stack Inventory Billing SaaS application built using React.js, Laravel REST API, Sanctum Authentication and MySQL.

## Features

- User Registration and Login
- Sanctum Token Authentication
- Role-Based Access: Admin, Manager, Customer
- Product Management
- Category Management
- Customer Management
- GST Invoice Generation
- Automatic Stock Deduction
- PDF Invoice Download
- Logo and Signature in Invoice PDF
- Dashboard Analytics
- Reports
- Business Settings
- Customer Invoice Portal
- Responsive Premium UI

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router
- Recharts
- jsPDF

### Backend
- Laravel
- Laravel Sanctum
- MySQL
- REST APIs
- Eloquent Relationships
